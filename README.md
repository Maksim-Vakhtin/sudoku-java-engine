# Sudoku Validation Engine & Game 🧩

**Description:** A fully playable Sudoku game operating natively in the command-line console. Built to demonstrate proficiency in complex data structures and algorithmic logic.

## 🛠️ Tech Stack
* **Language:** Java
* **Environment:** Eclipse IDE

## ✨ Key Features
* **Real-time Validation:** Writes highly optimized algorithms to strictly enforce standard Sudoku rules.
* **Complex Data Structures:** Utilizes 2D arrays to map and manipulate the game board.
* **Edge-case Handling:** Clean, encapsulated, and debuggable code structure.
